
\part{Fingerprints}

In diesem Abschnitt werden alle objektiven Daten unterteilt in Parts dargestellt. Ein Part besteht aus einer unterbrechungsfreien Serie von Daten. Sollte aus einem bestimmten Grund eine Unterbrechung des Datenflusses stattgefunden haben, so wurde anschließend ein neuer Part begonnen.\\
\\
Es werden nur Parts mit einer Mindestlänge von $minpartlength$ Minuten dargstellt. Daraus ergibt sich eine Anzahl von $numberofparts$ Parts.
